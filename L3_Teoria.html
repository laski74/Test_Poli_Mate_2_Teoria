$$(a \cdot b)^n = a^n \cdot b^n$$

La radice $n$-esima di un numero $a$ √® quel numero $b$ che elevato alla $n$ d√† $a$.

$$\sqrt[n]{a} = b \quad \Leftrightarrow \quad b^n = a$$

‚ö†Ô∏è Condizioni di esistenza:

$$\sqrt[n]{a} \cdot \sqrt[n]{b} = \sqrt[n]{a \cdot b}$$

2. Quoziente di radicali con lo stesso indice
$$\frac{\sqrt[n]{a}}{\sqrt[n]{b}} = \sqrt[n]{\frac{a}{b}}$$

$\frac{\sqrt{50}}{\sqrt{2}} = \sqrt{\frac{50}{2}} = \sqrt{25} = 5$
3. Potenza di un radicale
$$(\sqrt[n]{a})^m = \sqrt[n]{a^m}$$

$(\sqrt{3})^2 = 3$
$(\sqrt[3]{2})^6 = \sqrt[3]{2^6} = \sqrt[3]{64} = 4$
4. Radice di radice
$$\sqrt[m]{\sqrt[n]{a}} = \sqrt[m \cdot n]{a}$$

Esempio:

$\sqrt{\sqrt[3]{8}} = \sqrt[6]{8} = \sqrt[6]{2^3} = 2^{3/6} = 2^{1/2} = \sqrt{2}$

Un radicale si pu√≤ semplificare estraendo i fattori perfetti dal radicando.

  3. Porta fuori dalla radice i fattori completi

Si possono sommare o sottrarre solo radicali simili (stesso indice e stesso radicando).

Razionalizzazione

Razionalizzare significa eliminare i radicali dal denominatore di una frazione.

$$\frac{a}{\sqrt[n]{b}} = \frac{a \cdot \sqrt[n]{b^{n-1}}}{\sqrt[n]{b^n}} = \frac{a\sqrt[n]{b^{n-1}}}{b}$$

$$\frac{a}{b \pm \sqrt{c}} = \frac{a(b \mp \sqrt{c})}{b^2 - c}$$

Con lo stesso indice:

$$\text{Se } a < b \text{ allora } \sqrt[n]{a} < \sqrt[n]{b}$$

  2. $\sqrt[3]{5} = \sqrt[6]{5^2} = \sqrt[6]{25}$

  4. $25 < 27$ quindi $\sqrt[3]{5} < \sqrt{3}$
